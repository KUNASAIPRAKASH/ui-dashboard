# UI Dashboard – Next.js App Router

## Objective
Recreated the provided webpage design using Next.js

## Tech Stack
- Next.js (App Router)
- TypeScript (TSX)
- Tailwind CSS
- lucide-react
- next/navigation (useRouter)
- React useState

## Features
- Search functionality
- Column visibility toggle
- Status filter
- Pagination with boundary control
- Navigation using useRouter
- Custom favicon
- Modular reusable components

## Run Locally 
npm run dev

Open http://localhost:3000
